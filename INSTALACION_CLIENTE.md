# 🚀 Cómo instalar tu Sistema de Cuenta Corriente

## ⚡ Instalación súper rápida (5 minutos)

### 📋 Requisitos previos
Necesitas tener instalado **Node.js** en tu computadora:
- 🌐 Descárgalo gratis desde: https://nodejs.org
- ✅ Instala la versión **LTS** (recomendada)
- ⏱️ Toma 3 minutos instalarlo

### 📂 Pasos de instalación

#### 1. Descomprimir
- 📁 Descomprime el archivo ZIP que recibiste
- 📍 Puedes ponerlo en tu Escritorio o Documentos

#### 2. Abrir terminal
**En Windows:**
- 🔍 Presiona `Windows + R`
- ⌨️ Escribe `cmd` y presiona Enter
- 📂 Ve a la carpeta: `cd "ruta/donde/descomprimiste"`

**En Mac:**
- 🔍 Presiona `Cmd + Espacio`
- ⌨️ Escribe "Terminal" y presiona Enter
- 📂 Ve a la carpeta: `cd "ruta/donde/descomprimiste"`

#### 3. Instalar dependencias
```bash
npm install
```
⏳ *Espera 1-2 minutos mientras se instalan los archivos necesarios*

#### 4. Usar el sistema (¡SÚPER FÁCIL!)

**Opción 1 - UN SOLO CLIC (Recomendado):**
- 🖱️ **Doble clic en**: `Sistema_Cuenta_Corriente.vbs`
- ✨ **Se abre automáticamente** en tu navegador
- 🎉 **¡Listo para usar!**

**Opción 2 - Manual (si prefieres):**
- 🖱️ Doble clic en: `INICIO_RAPIDO.bat`
- 📱 Se abre terminal + navegador automáticamente

## ❓ Si algo no funciona

### Problema: "npm no reconocido"
**Solución:** Necesitas instalar Node.js desde https://nodejs.org

### Problema: Error al instalar dependencias
**Solución:**
1. Cierra terminal
2. Abre nueva terminal como administrador
3. Vuelve a ejecutar `npm install`

### Problema: Puerto ocupado
**Solución:**
- Cierra cualquier programa que use el puerto
- O ejecuta: `npm run dev:full -- --port 3001`
- Luego ve a: http://localhost:3001
